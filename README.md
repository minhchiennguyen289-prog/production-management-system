# Production Management System (ERP)

## 📌 Introduction
RDBMS system for manufacturing, featuring 3NF architecture and Python CLI integration.

## 📂 Structure
- `/database`: DDL/DML scripts.
- `/app`: Python source code & dependencies.
- `/docs`: Technical diagrams.

## 🚀 Setup
1. Import SQL scripts in `database/`.
2. Install dependencies: `pip install -r app/requirements.txt`.
3. Run: `python app/app.py`.
